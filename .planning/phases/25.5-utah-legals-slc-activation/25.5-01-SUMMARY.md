**One-liner:** Extended utah-legals.ts scraper with Salt Lake County at index 17, SLC parcel regex Branch 4, 84116 city allowlist filter, and zip='84116' threading through notice->upsert->normalizeCity() to produce Rose Park rows on the next scraper run.

### RP-10: Branch 4 Added to extractParcelId()

```typescript
// Branch 4: SLC multi-segment bare pattern (defensive fallback — Branch 2 catches labeled SLC parcels)
// Matches: 26-24-406-084-0000, 08-22-327-004-0000 (format: DD-DD-DDD-DDD-DDDD)
const slcParcel = text.match(/\b(\d{2}-\d{1,3}-\d{3,4}-\d{3,4}-\d{3,4})\b/);
if (slcParcel) return slcParcel[1];
```

Branches 1-3 unchanged. Branch 4 fires only when a bare SLC parcel number appears without a label prefix. Branch 2 (Serial No./Tax ID label) already catches the majority of labeled SLC parcels.

### RP-11: 84116 City Allowlist + zip Field Threading

Three sub-changes for RP-11:

**1. UtahLegalsNotice interface extended:**
```typescript
/** '84116' for SLC notices that pass the 84116 zip-area city allowlist; undefined otherwise */
zip?: string;
```

**2. SLC_84116_CITIES Set at module level:**
```typescript
const SLC_84116_CITIES = new Set(['salt lake city', 'rose park', 'north salt lake']);
```

**3. Allowlist guard in notice processing loop (after isForeclosure check):**
- Non-allowlist SLC cities: logged and `continue`d (Sandy, Midvale, etc. excluded)
- Empty city: logged with warning but allowed through (gets Rose Park COUNTY_CITY default)
- `zip: countyLower === 'salt lake' ? '84116' : undefined` set on notice object

**4. upsert.ts: zip threaded to upsertProperty():**
```typescript
const propertyId = await upsertProperty({
  parcelId,
  address: notice.propertyAddress ?? "",
  city,
  ownerName: notice.ownerName,
  zip: notice.zip,   // RP-11: '84116' for SLC notices, undefined for rural counties
}, county);
```

This activates the existing `normalizeCity()` chain: zip='84116' → city='Rose Park'.
