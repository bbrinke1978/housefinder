1. **scraper/src/db/schema.ts** — Add two new tables after the existing `scraperConfig` table:

   `ownerContacts` table:
   - `id`: uuid, defaultRandom, primaryKey
   - `propertyId`: uuid, notNull, references properties.id
   - `phone`: text (nullable)
   - `email`: text (nullable)
   - `source`: text, notNull — values: 'assessor' | 'truepeoplesearch' | 'manual'
   - `isManual`: boolean, notNull, default false
   - `needsSkipTrace`: boolean, notNull, default true
   - `createdAt`: timestamp with timezone, notNull, defaultNow
   - `updatedAt`: timestamp with timezone, notNull, defaultNow
   - Index: `idx_owner_contacts_property_id` on propertyId
   - Unique index: `uq_owner_contacts_property_source` on (propertyId, source) — for upsert deduplication

   `alertHistory` table:
   - `id`: uuid, defaultRandom, primaryKey
   - `leadId`: uuid, notNull, references leads.id
   - `channel`: text, notNull — values: 'email' | 'sms'
   - `runDate`: text, notNull — format 'YYYY-MM-DD'
   - `sentAt`: timestamp with timezone, notNull, defaultNow
   - Unique index: `uq_alert_history` on (leadId, channel, runDate)
   - Index: `idx_alert_history_lead_id` on leadId

2. **app/src/db/schema.ts** — Add the IDENTICAL `ownerContacts` and `alertHistory` table definitions (both apps share one PostgreSQL database). Copy exactly from scraper schema. Also add the `leadNotes` export if not already present (it should be there from Phase 2).

3. **scraper/tsconfig.json** — Add `"jsx": "react-jsx"` and `"jsxImportSource": "react"` to compilerOptions. Update the `include` array to also glob `"src/**/*.tsx"` files.
