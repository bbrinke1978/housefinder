<objective>
Extend the existing UGRC enrichment script (`app/src/scripts/import-ugrc-assessor.mjs`) so it can enrich Rose Park (Salt Lake County, zip 84116) properties via the pre-built parcel-ID allowlist (Option B from research). Add a `--dry-run` flag for safe shadow execution before any DB writes.

Purpose: Phase 26's foundational code change. Without this, no SLC enrichment is possible — the current script's `Parcels_SaltLake_LIR` is intentionally absent because the original `PARCEL_ZIP='84116'` filter doesn't exist in that ArcGIS layer. Research confirmed Option B (allowlist-driven) is the only viable approach: 42 ArcGIS batch requests vs. 395+ for full SLC, and ~16k DB queries vs. ~789k.

Output: Extended `import-ugrc-assessor.mjs` with a new `fetchFromAllowlist()` function, a new Salt Lake County entry in `COUNTIES`, and a `--dry-run` CLI flag. No DB migration required (all four columns are already nullable). No invocation in this plan — the run itself is Plan 02.
</objective>
