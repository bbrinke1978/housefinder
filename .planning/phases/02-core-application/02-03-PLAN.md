    1. Add to `app/src/lib/queries.ts`:

       **getPropertyDetail(propertyId: string)** — returns full property + lead data:
       - Inner join properties + leads on leads.propertyId = properties.id
       - Where properties.id = propertyId
       - Select all property fields + all lead fields (distressScore, isHot, status, lastContactedAt, etc.)
       - Return single result or null if not found

       **getPropertySignals(propertyId: string)** — returns DistressSignalRow[]:
       - Select from distressSignals where propertyId matches
       - Order by createdAt desc (newest first)

       **getPropertyNotes(leadId: string)** — returns LeadNote[]:
       - Select from leadNotes where leadId matches
       - Order by createdAt desc (newest first)

    2. Add to `app/src/lib/actions.ts` (create as "use server" file if not exists):

       **markLeadViewed(propertyId: string)** — server action:
       - Call auth() to verify session
       - Update leads set lastViewedAt = new Date() where propertyId matches
       - This clears the "new" badge on the dashboard for this property

       **addLeadNote(leadId: string, noteText: string)** — server action:
       - Call auth() to verify session
       - Validate with zod: leadId is uuid, noteText is non-empty string max 2000 chars
       - Insert into leadNotes with noteType = "user"
       - revalidatePath for the property detail page
  </action>
