**dashboard-filters.tsx — Mobile filter drawer:**

This is the critical mobile improvement. Currently 7 filter controls stack vertically on mobile, pushing property cards far down the page.

1. Import Sheet components: `import { Sheet, SheetContent, SheetHeader, SheetTitle, SheetTrigger, SheetFooter } from "@/components/ui/sheet"`
2. Import SlidersHorizontal icon from lucide-react
3. Create two layouts:

**Mobile layout (below md breakpoint):**
```tsx
<div className="flex gap-2 md:hidden">
  <div className="flex-1">
    {/* Search input — always visible */}
    <SearchInput />
  </div>
  <Sheet>
    <SheetTrigger asChild>
      <Button variant="outline" size="icon" className="relative flex-shrink-0">
        <SlidersHorizontal className="h-4 w-4" />
        {activeFilterCount > 0 && (
          <span className="absolute -top-1 -right-1 h-4 w-4 rounded-full bg-primary text-[10px] font-bold text-primary-foreground flex items-center justify-center">
            {activeFilterCount}
          </span>
        )}
      </Button>
    </SheetTrigger>
    <SheetContent side="bottom" className="h-[80vh] rounded-t-2xl">
      <SheetHeader>
        <SheetTitle>Filter Leads</SheetTitle>
      </SheetHeader>
      <div className="space-y-4 pt-4 overflow-y-auto flex-1">
        {/* All filter controls stacked vertically with labels */}
        <CitySelect />, <OwnerTypeSelect />, <DistressTypeSelect />,
        <TierSelect />, <HotLeadsToggle />, <SortSelect />
      </div>
      <div className="flex gap-2 pt-4 border-t border-border mt-4">
        <Button variant="outline" className="flex-1" onClick={clearAllFilters}>Clear All</Button>
```

**Desktop layout (md and above):**

**IMPORTANT:** The filter logic (URL searchParams, router.push, etc.) must NOT change. Only the visual layout changes. The Sheet is purely a mobile container — the same filter components render inside it.
