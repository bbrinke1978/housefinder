<task type="auto">
  <name>Task 1: Create batch geocoding script for existing properties</name>
  <files>app/src/scripts/geocode-properties.ts</files>
  <action>
  1. **Create `app/src/scripts/geocode-properties.ts`:**
     ```typescript
     /**
      * One-time batch geocoding script.
      * Reads all properties without lat/lng, geocodes via Mapbox API, updates DB.
      *
      * Usage: cd app && npx tsx src/scripts/geocode-properties.ts
      * Requires: NEXT_PUBLIC_MAPBOX_ACCESS_TOKEN environment variable
      */
     ```

  2. **Implementation:**
     - Import `db` from `@/db/client` and `properties` from `@/db/schema`
     - Import `eq, isNull, or` from `drizzle-orm`
     - Read `NEXT_PUBLIC_MAPBOX_ACCESS_TOKEN` from `process.env` — exit with error if missing
     - Query all properties where latitude IS NULL or longitude IS NULL:
       ```typescript
       const ungeocodedProps = await db.select({
         id: properties.id,
         address: properties.address,
         city: properties.city,
         state: properties.state,
         zip: properties.zip,
       }).from(properties)
         .where(or(isNull(properties.latitude), isNull(properties.longitude)));
       ```
     - If none found, log "All properties already geocoded" and exit
     - Log total count: `Found ${ungeocodedProps.length} properties to geocode`

  3. **Batch processing loop:**
     - Process in batches of 50
     - For each property, construct search query: `${address}, ${city}, ${state} ${zip || ""}`
     - Call Mapbox Geocoding API:
       ```typescript
       const url = `https://api.mapbox.com/geocoding/v5/mapbox.places/${encodeURIComponent(query)}.json?access_token=${token}&limit=1&country=US&types=address`;
       const res = await fetch(url);
       const data = await res.json();
       ```
     - Extract coordinates from `data.features[0].center` — returns `[longitude, latitude]`
     - If features array is empty or API error: log warning, increment fail counter, continue
     - Update the property:
       ```typescript
       await db.update(properties)
         .set({ latitude: lat, longitude: lng })
         .where(eq(properties.id, prop.id));
       ```
     - After each batch of 50, sleep 1 second: `await new Promise(r => setTimeout(r, 1000))`
     - Track and log progress: `Batch ${batchNum}: geocoded ${success}/${batchSize} (${totalDone}/${total} total)`

  4. **Final report:**
     ```
     Geocoding complete:
       Total: {total}
       Success: {successCount}
       Failed: {failCount}
       Skipped: {already had coords}
     ```
     Exit with code 1 if any failures, 0 if all succeeded.

  IMPORTANT: Use `tsx` to run this script (already a dev dependency via the scraper). The script uses TypeScript path aliases (@/), so use `npx tsx --tsconfig tsconfig.json src/scripts/geocode-properties.ts`.
  </action>
  <verify>Run `cd app && npx tsc --noEmit` to confirm script compiles. Verify the script file exists and imports db client correctly.</verify>

       ```tsx

         </div>
       </div>
       ```
     - Import `Search` from lucide-react
     - On Enter key press, call Mapbox Geocoding API:
       ```typescript
       async function handleSearch(e: React.KeyboardEvent) {
         if (e.key !== "Enter" || !searchQuery.trim()) return;
         const token = process.env.NEXT_PUBLIC_MAPBOX_ACCESS_TOKEN;
         const url = `https://api.mapbox.com/geocoding/v5/mapbox.places/${encodeURIComponent(searchQuery)}.json?access_token=${token}&limit=1&country=US&bbox=-114.05,-37.0,-109.04,42.0`; // Utah bounding box
         const res = await fetch(url);
         const data = await res.json();
         if (data.features?.[0]) {
           const [lng, lat] = data.features[0].center;
           mapRef.current?.flyTo({ center: [lng, lat], zoom: 14, duration: 2000 });
         }
       }
       ```
     - The `bbox` parameter limits search to Utah area for relevance
     - Use `mapRef` (useRef to the Map component) to call `flyTo` for smooth animation
