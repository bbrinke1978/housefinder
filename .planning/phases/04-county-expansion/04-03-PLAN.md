  <action>
  Create `scraper/src/sources/pdf-delinquent-parser.ts` â€” a shared module that all PDF-only counties use:

  1. **Define PdfCountyConfig type:**
     ```typescript
     export type PdfCountyConfig = {
       county: string;               // e.g., "sevier"
       treasurerPageUrl: string;      // Page to scrape for PDF link
       pdfLinkSelector: string;       // How to find the PDF link (CSS selector or text match)
       pdfLinkTextPattern: RegExp;    // Regex to match link text, e.g., /delinquent/i
       lineParser: (line: string) => DelinquentRecord | null;  // County-specific line parser
     };
     ```

  2. **Export `parsePdfDelinquent(config: PdfCountyConfig): Promise<DelinquentRecord[]>`:**
     - Step 1: Launch Playwright browser, navigate to `config.treasurerPageUrl`, wait for page load.
     - Step 2: Find all `<a>` elements on the page. Filter by `config.pdfLinkTextPattern` matching the link text AND href ending in `.pdf`. Take the first match. If no match, log `[{county}-delinquent-pdf] No PDF link found on treasurer page` and return [].
     - Step 3: Resolve the PDF URL (may be relative â€” resolve against page URL).
     - Step 4: Close browser. Download PDF via `fetch(pdfUrl)`. If response not OK, log and return [].
     - Step 5: Parse with `pdf-parse`: `import pdfParse from "pdf-parse"`. Extract `data.text`.
     - Step 6: Split text by newlines. For each line, call `config.lineParser(line)`. Collect non-null results.
     - Step 7: Validate each result with `delinquentRecordSchema.safeParse()`. Log invalid records. Return valid ones.
     - Log: `[{county}-delinquent-pdf] Parsed {N} records from PDF`

  3. **Export per-county configs as constants:**

     **sevierConfig:**
     - treasurerPageUrl: `https://www.sevier.utah.gov/departments/county_officials/treasurer/current_year_delinquent_tax_report.php`
     - pdfLinkTextPattern: `/delinquent.*tax/i`
     - lineParser: Parse lines matching parcel pattern. Sevier parcel format is typically `XX-XXXX-XXXX` or similar. Parse: parcelId, ownerName, amountDue from whitespace-separated fields. Use a generous regex that captures parcel-like patterns at line start.

     **juabConfig:**
     - treasurerPageUrl: `https://juabcounty.gov/` then navigate to treasurer page (check for /treasurer/ or /departments/ link)
     - pdfLinkTextPattern: `/delinquent/i`
     - lineParser: Same general pattern as Sevier. Juab parcels may be formatted differently.

     **millardConfig:**
     - treasurerPageUrl: `https://millardcounty.gov/` then navigate to treasurer page
     - pdfLinkTextPattern: `/deli[nq]*uent/i` (note: research showed "Deliquent" typo in Millard's URL â€” match both spellings)
     - lineParser: Same general pattern.

     **sanpeteConfig:**
     - treasurerPageUrl: `https://sanpetecountyutah.gov/` then navigate to treasurer page
     - pdfLinkTextPattern: `/delinquent/i`
     - lineParser: Same general pattern.

  4. **Generic lineParser factory:** Since the exact PDF format for each county is not verified (MEDIUM confidence from research), create a `makeGenericDelinquentLineParser(county: string)` function that:
     - Skips blank lines and header-like lines (lines containing "PARCEL" or "OWNER" as header text).
     - Tries to match a parcel number pattern at the beginning of the line: `/^(\d{1,3}[-\s]\d{3,5}[-\s]\d{3,5})/.
     - Extracts remaining text as ownerName (everything after parcel up to a dollar amount).
     - Extracts dollar amount if present: `/\$?([\d,]+\.?\d*)$/`.
     - Returns `{ parcelId, ownerName, amountDue, county }` or null if no parcel found.
     - This is intentionally generous â€” better to parse too many records (and let validation filter) than miss data.

  IMPORTANT: Use `.js` extensions on all imports. Import `pdfParse` as default import: `import pdfParse from "pdf-parse"`.
