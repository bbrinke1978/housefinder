# Code Violation Publications Research
# Rural Utah Counties: Carbon, Emery, Juab, Millard, Sanpete, Sevier

**Researched:** 2026-03-19
**Research Mode:** Feasibility / Ecosystem
**Overall Confidence:** HIGH (direct website verification for most findings)

---

## Executive Summary

None of the six target counties (Carbon, Emery, Juab, Millard, Sanpete, Sevier) publish a publicly accessible,
searchable online database of code enforcement violations. These are small rural Utah counties where code
enforcement is handled at the county or city level with minimal digital infrastructure.

The landscape is consistent across all six counties:

- Code enforcement exists (officers are appointed, ordinances are on the books)
- Violations are tracked internally, not published online
- Permit applications are moving online via CityInspect (used by Carbon, Sanpete, Sevier)
- The closest thing to a public record is Utah's court system (XChange) for cited violations
- GRAMA (Utah's open records law) is the mechanism to request records — no proactive online publication

The most actionable data path for HouseFinder is **Utah Courts XChange** (paid subscription), which covers
justice court ordinance violations. County-level administrative enforcement (nuisance letters, abatement orders
that never go to court) has no public online equivalent in these counties.

## State-Level Mechanisms

### Utah Courts XChange

**URL:** https://xchange.utcourts.gov/

**What it covers:**
- District court and justice court case information statewide
- Justice courts handle: ordinance violations, class B/C misdemeanors, infractions, traffic cases
- All six target counties have justice courts that handle local ordinance violations

**Searchable fields:**
- Jurisdiction (District or Justice Court)
- Case number
- Party name (last name + first name, with wildcard support)
- Case type
- Date of birth
- Citation number
- Search scope: statewide, by county, or specific court

**Can filter by:**
- County
- Court type (Justice Court)
- Case type (ordinance violation is a listed category)
- Date range (filing date, disposition date, judgment date)

**Cost:**
- Guest Account: $5 setup fee + $0.20/search + $0.50/document + $1.00/document image
- Monthly Subscription: $25 setup + $40/month (500 searches included)
- Free public terminals at most district court locations

**Key limitation:** XChange covers cases that have been filed in court. Administrative enforcement actions
(warning letters, abatement orders, civil penalties that don't go to court) are not in this system.

**Scrapability:** LOW — requires paid account, has CAPTCHA, is not a public API. Would require Playwright
automation with a paid XChange account.

### Utah Public Notice Website (PMN)

**URL:** https://www.utah.gov/pmn/

**What it covers:**
- Public meeting notices, agenda items, and public hearing notices
- Code enforcement hearings for zoning violations may be posted here
- Sanpete County posts administrative code enforcement hearing notices on PMN

**Searchable by:**
- Entity type and name
- Public body name
- Notice title/agenda keywords
- Date ranges
- Tags

**Scrapability:** MEDIUM — HTML-based search interface, potentially scrapable but not structured violation data.
Contains hearing notices, not violation lists.

### Utah Building Permits Data

**URL:** https://secure.utah.gov/datarequest/buildingpermits/index.html

**What it covers:** Building permits issued statewide (not violations)

**Scrapability:** HIGH — state portal with downloadable data, but permits ≠ violations.

## GRAMA (Utah Open Records Law)

Utah's Government Records Access and Management Act (GRAMA) gives the public the right to request
code enforcement records from any county or city. This is the mechanism for accessing violation records
that are not published online.

**Process:**
1. Submit written GRAMA request to county/city clerk
2. Records must be provided within 10 business days
3. Fee: No charge for inspection; copying fees apply
4. Some enforcement records may be classified (if active investigation)

**GRAMA for bulk/ongoing access:** Not practical for scraping — each request is manual, has response delays,
and fees for copying. Not a viable data source for automated lead generation.

## Recommendations for HouseFinder

### What to pursue

1. **Utah Courts XChange** — The most structured public data source for code violations that escalated to
   court citations. Covers all six counties' justice courts. Requires a paid subscription ($40/month).
   Can be searched by county + case type (ordinance violation). Scrapable with Playwright + paid account.
   Limitation: only captures court-filed violations, not administrative enforcement.

2. **Meeting minutes scraping** — County and city council/P&Z meeting minutes are published as PDFs on
   county/city websites. Code enforcement actions are often discussed in these minutes. Low structure,
   requires NLP/LLM parsing. Examples: Price City P&Z minutes at `priceutah.gov`.

3. **Utah PMN (Public Notice Website)** — Sanpete County and likely others post administrative code
   enforcement hearing notices on `utah.gov/pmn/`. Scrapable HTML search interface. Contains hearing
   notice content that may include property addresses and violation types.

4. **County Recorder filings** — Some counties (like North Salt Lake as a model) file Notices of Violation
   with the County Recorder, making them part of the property record chain. Worth checking each county
   recorder's online search for "notice of violation" or "nuisance abatement" documents.
   Sevier County recorder: `qdocs.sevier.utah.gov/recorder/web/`

### What NOT to pursue

- Direct scraping of CityInspect portals — login-gated, no public API
- GRAMA requests for bulk data — too manual and slow for ongoing lead generation
- Building permit data — tracks construction, not violations

### Data quality expectation

Even with XChange, expect LOW volume from these rural counties. These are very small populations:
- Carbon County: ~20,000 residents
- Emery County: ~11,000 residents
- Juab County: ~12,000 residents
- Millard County: ~13,000 residents
- Sanpete County: ~32,000 residents
- Sevier County: ~22,000 residents

Code enforcement citation rates in rural Utah are likely 10-50 cases/year per jurisdiction. XChange would
be the most reliable source but yield modest data.
